= ONTAP Select 비디오 콘솔에 접속하세요
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP Select 실행 중인 하이퍼바이저 가상 머신의 비디오 콘솔에 액세스할 수 있습니다.

.이 작업에 관하여
문제를 해결하거나 NetApp 지원팀에서 요청할 때 가상 머신 콘솔에 액세스해야 할 수도 있습니다.

.단계
. vSphere 클라이언트에 액세스하여 로그인합니다.
. 계층 구조에서 적절한 위치로 이동하여 ONTAP Select 가상 머신을 찾습니다.
. 가상 머신을 마우스 오른쪽 버튼으로 클릭하고 *콘솔 열기*를 선택합니다.
