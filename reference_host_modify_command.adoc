= ONTAP Select Deploy에 대한 호스트 관리 서버 수정
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
당신은 사용할 수 있습니다 `host modify` ONTAP Select .



== 통사론

`host modify [-help] [-foreground] -name _name_ -mgmt-server _management_server_ [-username _username_]`



== 필수 매개변수

[cols="35,65"]
|===


| 매개변수 | 설명 


 a| 
`-name _name_`
 a| 
수정하려는 호스트의 IP 주소 또는 FQDN입니다.



 a| 
`-mgmt-server _management_server_`
 a| 
호스트에 설정할 호스트 관리 서버의 IP 주소 또는 FQDN입니다. 호스트에서 관리 서버의 설정을 해제하려면 "-"(하이픈)를 지정하세요. 이 호스트를 등록하기 전에 이 관리 서버에 대한 자격 증명을 추가해야 합니다. `credential add` 명령.

|===


== 선택적 매개변수

[cols="35,65"]
|===


| 매개변수 | 설명 


 a| 
`-help`
 a| 
도움말 메시지를 표시합니다.



 a| 
`-foreground`
 a| 
이 매개변수는 장기 실행 명령의 동작을 제어합니다. 이 매개변수를 설정하면 명령이 포그라운드에서 실행되고 작업과 관련된 이벤트 메시지가 발생하는 즉시 표시됩니다.



 a| 
`-username _username_`
 a| 
이 호스트에 액세스할 수 있는 사용자 이름입니다. 호스트가 관리 서버(즉, vCenter에서 관리하는 ESX 호스트)에서 관리되지 않는 경우에만 필요합니다.

|===
